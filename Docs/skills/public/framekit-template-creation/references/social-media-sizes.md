# Social Media Template Sizes

Quick design reference for Facebook, Instagram, TikTok, YouTube, X, and
LinkedIn, based on practical targets for July 2026. Use these as design
templates, not as a list of every ratio a platform might technically accept.

## Master Formats

- **Square feed:** `1080 x 1080` (`1:1`). Use for universal posts, catalogs,
  ad carousels, community posts, and mixed-platform image sets.
- **Portrait feed:** `1080 x 1350` (`4:5`). Best shared portrait format for
  Facebook, Instagram, and LinkedIn; use for mobile feed graphics and ads.
- **Instagram portrait:** `1080 x 1440` (`3:4`). Best for Instagram-first
  photos, portfolios, and the current profile grid.
- **Full-screen vertical:** `1080 x 1920` (`9:16`). Use for Stories, Reels,
  Shorts, TikTok, and immersive mobile video.
- **Social landscape:** `1200 x 630` (`1.91:1`). Use for link previews, Open
  Graph images, and wide social cards.
- **Landscape video:** `1920 x 1080` (`16:9`). Use for YouTube, events, live
  streams, webinars, interviews, and traditional video.
- **Avatar master:** `1080 x 1080` (`1:1`). Use for logos and profile images;
  export smaller copies from this master.

For advertising, prepare `1:1`, `4:5`, and `9:16` exports. Do not solve these
formats with an automatic crop when text, faces, or calls to action matter.

## Platform Guide

### Facebook

- **Identity:** Page profile `800 x 800`; Page cover `851 x 315` (or `1702 x
  630`); Group cover `1640 x 856`; Event cover `1920 x 1005`.
- **Feed:** Prefer `1080 x 1350` for portrait posts, `1080 x 1080` for the
  safest reusable format, and `1200 x 630` for links or wide images.
- **Carousels:** Keep every image square or every image `4:5`; mixed ratios are
  likely to be cropped inconsistently. Use `1080 x 1080` for the safest ad
  carousel.
- **Stories and Reels:** Use `1080 x 1920`. Keep important content away from
  the top and bottom interface areas.
- **Traditional video:** Use `1920 x 1080`; use `1080 x 1350` for vertical Feed
  video that is not a Reel.

### Instagram

- **Identity:** Profile master `1080 x 1080`; practical minimum `320 x 320`.
  Keep the mark centered because it is displayed as a circle.
- **Feed:** Prefer `1080 x 1440` (`3:4`) for Instagram-first work. Use `1080 x
  1350` (`4:5`) when the design must also work on Facebook or LinkedIn. Use
  `1080 x 1080` for catalogs and reusable cards.
- **Landscape posts:** Use `1080 x 566` only when the horizontal composition is
  essential; it uses less space in the mobile feed.
- **Carousels:** Use one ratio for every slide. Choose `3:4` for Instagram-first
  storytelling, `4:5` for cross-platform graphics, or `1:1` for compatibility.
  Instagram supports up to 20 photos or videos.
- **Stories and Reels:** Use `1080 x 1920`. Reel covers can use approximately
  `1080 x 1682`.

### TikTok

- **Profile:** Use a centered square avatar, typically `400 x 400` or an export
  from the `1080 x 1080` master.
- **Video and Stories:** Use `1080 x 1920`; this is TikTok's primary format.
- **Photo posts:** Use `1080 x 1920` for full-screen sequences. Keep all slides
  consistent; a post can contain up to 35 photos.
- **Square or landscape:** `1080 x 1080` and `1920 x 1080` are accepted for
  reuse, but they do not fill a phone screen as effectively.
- **Covers:** TikTok usually selects a video frame instead of requiring a
  separate cover upload. Design a readable cover moment into the video.
- **Ad carousels:** Prefer `1080 x 1920`; the practical vertical minimum is
  `720 x 1280`. Use `1200 x 628` only for horizontal campaigns.

### YouTube

- **Channel identity:** Profile `800 x 800`; banner `2560 x 1440`; keep text
  and logos inside the approximate safe area `1544 x 423`.
- **Watermark:** Use a simple square mark of at least `150 x 150` and under
  `1 MB`.
- **Long-form video:** Use `1920 x 1080` for Full HD or `3840 x 2160` for 4K.
- **Thumbnails:** Current guidance favors `3840 x 2160`; older workflows use
  `1280 x 720`. Keep the thumbnail readable at a small mobile size.
- **Shorts:** Use `1080 x 1920`. The cover is selected from the video, so design
  a strong cover frame.
- **Community:** Use `1080 x 1080`; square images are safest for posts and polls.
- **Podcast:** Use a square `1280 x 1280` cover.

### X

- **Identity:** Profile `400 x 400`; header `1500 x 500`. Keep the header's
  important content clear of the profile image and edges.
- **Images:** Use `1600 x 900` for common landscape posts, `1200 x 1600` for
  tall images, `1200 x 1200` for square posts, and `1200 x 628` for link cards.
- **Multiple images:** A post supports up to four photos. Use identical square
  images when predictable mosaic cropping matters most.
- **Video:** Use `1920 x 1080` for horizontal video. Use `1080 x 1800` (`3:5`)
  as a conservative vertical upload instead of relying on full `9:16` support.

### LinkedIn

- **Identity:** Personal profile and company logo `400 x 400`; personal cover
  `1584 x 396`; company cover `4200 x 700`.
- **Other surfaces:** Event image `1280 x 720`; newsletter logo `300 x 300`;
  article or newsletter cover `1920 x 1080`.
- **Feed:** Prefer `1080 x 1350` for mobile posts, announcements, case studies,
  hiring graphics, and educational content. Use `1080 x 1080` for reusable
  cards and `1200 x 627` for links.
- **Image sets:** Keep all images at `1080 x 1350`; LinkedIn supports up to 20
  photos and the first image strongly affects the preview.
- **Document carousels:** LinkedIn's organic carousel is a PDF or document,
  not a native image carousel. Design `1080 x 1350` pages and export them as a
  PDF. Use `1080 x 1080` for a more reusable square document.
- **Video:** Use `1080 x 1350` for Feed, `1080 x 1920` for full-screen vertical,
  and `1920 x 1080` for interviews, demos, and webinars.

## Shared Rules

- Use `3:4` for Instagram-first feed content, `4:5` for cross-platform portrait
  content, `1:1` for universal cards and carousels, `9:16` for full-screen
  content, `1.91:1` for link previews, and `16:9` for traditional video.
- Keep every slide in a carousel at the same dimensions and aspect ratio.
- In `1080 x 1920` designs, keep critical content out of roughly the top `250`
  px and bottom `350` px; captions and controls may cover those areas.
- Center logos and faces inside avatars because profile images are often circular
  and displayed very small.
- Use separate banner templates. Their extreme ratios cannot be reliably made
  from a feed post.
- Prefer one content system with export presets. Create separate artwork when
  the crop changes the reading order, hierarchy, or safe area.
