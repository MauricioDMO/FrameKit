# @mauriciodmo/create-framekit

Scaffold a new FrameKit project with one command:

```bash
# After npm publication is confirmed:
pnpm dlx @mauriciodmo/create-framekit my-project
cd my-project
pnpm dev
```

The creator requires **pnpm** and exactly one nonexistent target directory.
It installs dependencies and generates the initial template catalog automatically.
If installation or generation fails, the partially-created project is preserved for diagnosis.

For template authoring patterns, see the [Template Authoring Guide](https://github.com/MauricioDMO/FrameKit/blob/main/Docs/en/guides/template-authoring.md).
