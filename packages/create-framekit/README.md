# @mauriciodmo/create-framekit

Crea un proyecto FrameKit listo para editar:

```bash
pnpm dlx @mauriciodmo/create-framekit mi-proyecto
cd mi-proyecto
pnpm dev
```

El creador instala las dependencias y genera el catálogo inicial de plantillas.
Para diseños complejos, consulta el patrón extraído con
[`defineTemplateBase`](https://github.com/MauricioDMO/FrameKit/blob/main/packages/framekit/README.md#extracted-artwork).
